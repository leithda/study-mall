



## 创建项目结构

创建如下项目结构

```bash
├─doc # 存放Markdown文件
└─study-mall # 后台项目
    ├─study-mall-common # 公共模块
    ├─study-mall-gateway # 网关模块
    ├─study-mall-service # 服务实现
    ├─study-mall-service-api # 服务接口及实体类
    └─study-mall-third # 第三方服务聚合模块
```
